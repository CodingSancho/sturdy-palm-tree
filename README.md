# Incremental Game

Unity version: 6000.3.11f1

Prototype

A single button that increments the score when clicked.

Score displayed using a number with dynamic formatting (e.g., 1k, 1M).

Future Plans

Auto-generators: passive income that increases your score automatically.

Upgrades: buttons that increase auto-increment rate or other abilities with scalable costs.

Save system: store progress locally so your game state persists between sessions.

UI polish: cozy background, animations, and improved visuals for a more immersive experience.
